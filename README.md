# Back-end Node.js
Teste Eyemobile

------------------------------------------
# Requisitos para a instalação:

* Node.js v14.+
* MySQL v8.+

------------------------------------------

# Instalação Back-end:

Execute na pasta raiz do projeto o comando (CMD):
```
 npm install
```

Aguarde a instalação e em seguida execute o comandos:
```
 npm start
```
Será exibido uma mensagem em caso de sucesso, basta acessar a URL: 
```
 http://localhost:3333
```
